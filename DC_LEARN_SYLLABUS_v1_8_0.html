02 — Learning method
Six tabs. One engineering system. Fully explained.

Each module has six tabs. The Trivium (Grammar → Logic → Rhetoric) is the core teaching structure inside the Learn tab. The Visual Guide, Field Challenges, and Assessment tabs complete the learning cycle — from understanding, to seeing, to applying, to being tested.

1
System Overview — the chain at a glance
The opening tab maps the full engineering chain — nine levels from first principles to investment committee language. The Cascade Calculator runs live bottleneck detection: enter target rack density and PUE, and every level shows PASS or FAIL. Click any level to read the plain English summary and open it in the Learn tab.
9-level chain map Cascade calculator PASS / FAIL detection Progress dots per stage
2
Learn — the Trivium in three stages
Grammar defines every term three ways — plain English, by the numbers, and technical definition with standards references. The Feynman Clarify button cycles through analogy, first principles, and a self-test. Logic builds cause-and-effect chains with Weakest Link analysis. Rhetoric shows how Declan, Ann, Mark, Sarah, and Tom each see the same engineering fact differently.
Feynman Clarify Cause & effect chains Weakest Link (ToC) 5 persona perspectives
3
Visual Guide — how it actually works
3–5 interactive engineering diagrams per module showing system states, failure modes, and design logic step by step. Each diagram passes the Feynman Test: it explains something that text alone cannot. Step through a UPS failure. Watch a free cooling switchover. See what happens when hot aisle containment fails.
Step-based walkthroughs Failure mode visualisation Feynman-tested diagrams 3–5 diagrams per module
4
Field Challenges — apply it at Clonshaugh
Nine engineering challenges per module, one per level. Each challenge puts you on site at Clonshaugh with a specific problem to solve — real parameters, real numbers, real constraints. A Show the Fix button reveals the step-by-step worked answer with sourced numbers and persona perspective. These are not textbook questions. They are the questions a 10-year DC professional would actually face.
9 challenges per module Clonshaugh scenarios Worked answers with sources Persona context
5
Assessment — three tiers of testing
Tier 1 Knowledge Check tests factual recall — definitions, thresholds, standards. Tier 2 Calculation tests whether you can work through the numbers (PUE, energy, cost, payback). Tier 3 Professional Judgement tests decision-making under constraint. Score 80% overall for a Completion certificate. Score 90% for Achievement.

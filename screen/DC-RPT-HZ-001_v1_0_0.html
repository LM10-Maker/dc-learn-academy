Weighted PUE Analysis

4.1Current State

Hall A: 1.2 MW IT at PUE 1.45 = 1.74 MW facility energy. Hall B: 1.2 MW IT at PUE 1.55 = 1.86 MW facility energy. Total: 3.60 MW facility / 2.4 MW IT.

Weighted PUE = 3.60 / 2.4 = 1.500. Above EU Taxonomy threshold.

4.2Target State

Hall A: 3.0 MW IT at PUE 1.20. Hall B: 1.2 MW IT at PUE 1.35. Total IT: 4.2 MW.

Weighted PUE = (3.0×1.20 + 1.2×1.35) / 4.2 = 5.22 / 4.2 = 1.243. EU Taxonomy: PASS.

CRITICAL CONSTRAINT — MIC Exceedance: Target facility load 5.22 MW / PF 0.95 = 5.49 MVA. Exceeds 5 MVA MIC by 0.49 MVA (9.9%). ESB Networks MIC increase application required before Hall A conversion. Binding design target for Hall A: PUE ≤ 1.28. Verified: (1.3 × 4.2 − 1.2 × 1.35) / 3.0 = 1.28.
